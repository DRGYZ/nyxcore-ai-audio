# Screenshot Checklist

Place showcase screenshots for the README and portfolio demos in this folder.

Preferred filenames:

- `dashboard.png`
- `review-inbox.png`
- `history.png`
- `saved-playlists.png`

Capture checklist:

- use the local web UI with the NyxCore demo library or a similarly safe sample set
- show live API mode, not mock fallback, when possible
- keep the browser width consistent across captures
- avoid personal file paths or machine-specific identifiers if you plan to publish the images
- refresh the demo reports before capturing so counts match the current README walkthrough
